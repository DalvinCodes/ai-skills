# AI Skills Repository

A collection of AI-agnostic skills for use with various AI coding assistants and agents.

## Structure

This repository contains reusable skill definitions that can be used across different AI platforms.

### Agent Skills
- `bastion/` - Autonomous agent delegation patterns with milestone-gated oversight
- `cs-c-level/` - C-level advisor skills (CEO, CTO)
- `cs-design-director/` - Design director skills for visual strategy
- `cs-marketing/` - Marketing skills for demand generation and content
- `cs-product/` - Product management skills for prioritization and discovery

### Development Skills
- `devops-claude-skills/` - DevOps workflows and best practices (GitOps, K8s, Terraform)
- `frontend-design/` - Frontend design patterns and guidelines
- `go-development/` - Go programming best practices
- `phaser-gamedev/` - Phaser 3 game development patterns
- `remotion-best-practices/` - Remotion video creation in React

## Usage

To use these skills with your AI assistant, copy the relevant skill folder to your AI's skills directory.

Each skill contains a `SKILL.md` file with detailed instructions on:
- What the skill does
- When to use it
- How to invoke it
- Required parameters and outputs

## Contributing

When adding new skills:
1. Create a new directory with a descriptive name
2. Include a `SKILL.md` file describing the skill's purpose and usage
3. Follow the existing directory naming conventions (kebab-case)
4. Include any supporting files (prompts, examples, references) in the directory
